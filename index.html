Graphien

Duo-lingo for learning how to write

Graphien, which is "to write" in Greek, was our submission @BayHacks2024 held at Tampa Bay that was a "DuoLingo" for learning how to write. Built in less than 30 hours, we leveraged a CNN model fine-tuned to free-form drawings of handwritten english alphabets.We emerged as winners :)

AutoWise

Smart Used Car Market Place

Complete used car market place product that I built for my data mining class. Data & images for this project was scraped from Vroom.com using selenium. I integrated a recommender system for similar cars based on current viewing car using a similarity model pre-computed and stored on a Postgres Sever. I also made a price prediction model which gives you optimal price to sell based on a Machine Learning Algorithm.

Handgun Detection - Security Footage

Object Detection - Computer Vision

The idea was to build an object detector model to detect Handguns from surveillance footage. Trained a YOLO-v5 model on over 50,000 handguns - taken from ROBOFLOW. This implmentation allows you to upload images, videos and it gives a downloadable file with a bounding box of the detection.
